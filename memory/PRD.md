# Wedding Website Template - PRD

## Original Problem Statement
Svatební šablona pro prodej na Etsy. Klíčové požadavky:
- Modernizovaný design s 5 barevnými tématy
- Ochrana heslem
- Veškerá konfigurace v jednom souboru
- PDF návod pro úplné začátečníky
- Port 3001 pro localhost

## Tech Stack
- **Frontend:** React 18, Vite 5
- **Styling:** Tailwind CSS 3
- **Animations:** Framer Motion
- **Testing:** Vitest, React Testing Library
- **Port:** 3001

## ✅ Completed Features (Feb 2025)
- [x] Plně responzivní design
- [x] Ochrana heslem
- [x] 5 barevných témat (sage, blush, dustyBlue, gold, lavender)
- [x] Dynamický přepínač témat
- [x] Centralizovaná konfigurace v `wedding.config.js`
- [x] Smooth scroll navigace
- [x] Framer Motion animace
- [x] 51 automatizovaných testů
- [x] ReadMe.pdf s detailním návodem pro začátečníky (12KB, 10 stran)
- [x] Port změněn na 3001

## Key Files
- `/app/src/config/wedding.config.js` - Hlavní konfigurační soubor
- `/app/ReadMe.pdf` - Uživatelský návod
- `/app/vite.config.js` - Konfigurace serveru (port 3001)
